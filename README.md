# 🚀 Omar Mostafa

منصة تعليمية متكاملة لطلاب المرحلة الثانوية لتعلم مادة الفيزياء بأسلوب حديث ومبتكر تحت إشراف الأستاذ عمر مصطفي.

---

## 🎯 فكرة المشروع
منصة Omar Mostafa تهدف لتبسيط الفيزياء من خلال:
- اختبارات إلكترونية تفاعلية.
- مساعد ذكي (تسلا) للطالب لحل المسائل والشرح اللحظي.
- مساعد ذكي (تسلا) للأستاذ لتحليل البيانات وإدارة المحتوى.
- كورسات تعليمية منظمة لكل الصفوف.
- لوحة تحكم ذكية للأستاذ لمتابعة أداء الطلاب.

---

## ⚙️ التقنيات المستخدمة
- **Next.js & React**: لبناء واجهة سريعة وسلسة.
- **Firebase**: لإدارة قاعدة البيانات والمصادقة.
- **Genkit**: لتشغيل المساعدين الأذكياء وتوليد الأسئلة.
- **Tailwind CSS**: لتصميم عصري ومتجاوب.

---
