# **App Name**: ExamPrep

## Core Features:

- Student Authentication: Allow students to create accounts and log in securely to access exams.
- Exam Creation: Enable admin users to create and configure exams with questions, answers, and scoring.
- Exam Listing: Display available exams for students to choose from.
- Exam Taking: Allow students to take exams online with a timer.
- Automated Grading: Automatically grade the exam after submission.
- Score Reporting: Show each student's score after they finish the exam. 
- Admin Dashboard: Provide an admin dashboard to manage exams, students, and view overall performance. It shows individual student grades and total scores for each student.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for a calm, focused environment.
- Background color: Light gray (#F0F8FF), very lightly tinted towards the hue of sky blue.
- Accent color: Sea green (#3CB371) for positive feedback and action buttons.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable interface.
- Use clear, outline-style icons for navigation and exam controls.
- Maintain a clean and intuitive layout to minimize distractions.
- Subtle animations and transitions for a smooth user experience.
